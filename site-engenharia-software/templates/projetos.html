Projetos realizados e tecnologias aplicadas

Durante minha formação técnica e estudos pessoais, venho desenvolvendo projetos com o objetivo de aplicar os conhecimentos adquiridos e criar soluções úteis com diversas linguagens de programação e frameworks.

Linguagens utilizadas:

  • Python: Programas com Tkinter, automações, Flask para web apps.
  • Java: Lógica orientada a objetos, interfaces gráficas, estrutura de dados.
  • C++: Estrutura de algoritmos, vetores, ponteiros e cálculo de desempenho.
  • JavaScript: Interatividade para páginas web, junto com HTML e CSS.
  • HTML/CSS: Criação de interfaces visuais com estrutura e estilo profissional.

Exemplos de projetos:

  • Aplicativo de cálculo de IMC em Python
  • Site com Flask para apresentar minha carreira (este projeto)
  • Interface em Java com botão e entrada de dados
  • Simulador de operações matemáticas em C++
  • Página web com menu interativo usando JavaScript

Importância dos projetos

Projetos práticos me permitem aplicar teoria, aprender com erros, montar portfólio e me preparar melhor para o mercado de trabalho. Além disso, mostram para as empresas que estou comprometido com minha evolução técnica.
